Getting started

Social buttons uses Polymer library for all components, then you need to import it to your site before start to use Social buttons.

  1. Import Polymer to your site: <script src="//cdnjs.cloudflare.com/ajax/libs/polymer/0.0.20131107/polymer.min.js"></script>
  2. Import Social Buttons to your site by:
    • Clonnig our git repo with git clone git://github.com/maracaipe/social-buttons.git
    • Download latest release (not available yet; you can download the working branch gh-pages here, instead.)
    • Installing our bower package (1.0 release) with bower install <not-yet>
  3. Put social buttons to your site with link import: <link rel="import" href="dist/<module>.html">
    • Load all components by pointing all.html
    • instead, you can load a component you want by pointing <social-network>/<component>.html
  4. Start to use social button components!

Usage

Put a like button on the page by placing <facebook-like></facebook-like> to your code.

Options

Parameter Values Default Description
action like, recommend like

The verb to display on the button.
colorscheme light, dark light

The color scheme used on the button.
href any absolute url current url

The absolute URL of the page that will be liked.
layout standart, button_count, box_count button_count

Selects one of the different layouts that are available for the plugin.
ref None

A label for tracking referrals which must be less than 50 characters and can contain alphanumeric characters and some punctuation (currently +/=-.:_).

share true, false true

Specifies whether to include a share button beside the Like button.
show_faces true, false false

Specifies whether to display profile photos below the button (standard layout only).
width some width 120(px)

The width of the plugin. The layout you choose affects the minimum and default widths you can use.
